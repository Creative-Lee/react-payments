## 어플리케이션 실행 방법!

### 1. 패키지 다운받기

```bash
npm i
```

위 명령어로 필요한 패키지를 다운로드해 주세요!

### 2. 프로젝트 실행

```bash
npm run start
```

위 명령어를 터미널에 입력해주세요!

### 3. 카드 등록😀

```
카드 추가 버튼을 눌러 새로운 카드를 등록하세요!
카드사 선택 - 필수 정보 입력 - 별칭 입력 순으로 이어집니다.
```

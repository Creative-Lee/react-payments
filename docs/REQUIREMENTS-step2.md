# 레벨2 - payments 미션 - step2 💰

## 기능 구현 목록

- 카드사 선택 기능
  - 카드사 선택 모달
    - [x] 모달에서 카드사 선택을 할 수 있다.
    - [x] 카드 미리보기 클릭 시 카드사 선택 모달을 다시 열 수 있다.
  - [x] 카드 미리보기에 선택한 카드사를 보여준다.
  - [ ] 각 카드사 시그니처 색상을 카드 미리보기에 보여준다.
- 등록 완료 페이지
  - [ ] 완성된 카드 미리보기를 보여준다.
  - [ ] 카드 별칭을 저장할 수 있다.
- 보유 카드 목록 페이지
  - [ ] 보유 카드 하단에 별칭을 보여준다.
